# Dynamic Clinical Pathway Generator

An AI-assisted clinical decision support system that dynamically generates patient-specific care workflows and explains clinical reasoning in plain language.

## Features
- Rule-based clinical pathway generation
- LLM-powered explanation layer (Groq)
- Flowchart visualization
- Safety-first, non-diagnostic design

## Tech Stack
- Backend: Python, Flask
- Frontend: React, Vite
- LLM: Groq (LLaMA 3.1)

## Setup

### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm run dev
